# Aero-Dashboard

## Описание

Aero-Dashboard - это мощный инструмент для мониторинга и визуализации данных, созданный для отслеживания ключевых показателей производительности и отображения информации в реальном времени.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/<ваш_пользователь>/aero-dashboard.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd aero-dashboard
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```

## Использование

### Запуск приложения
Для запуска приложения используйте следующую команду:
```bash
npm start
```

### Доступные маршруты
- `/api/data`: Получить данные в реальном времени.
- `/api/stats`: Получить статистику.
