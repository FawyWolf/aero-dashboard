# Contribution Guidelines

Thank you for considering contributing to the **aero-dashboard** project! We appreciate your efforts and want to ensure a smooth collaboration process. Please follow the guidelines below:

## Bug Reporting
1. **Check for existing issues**: Before reporting a new bug, please check the existing issues to see if your problem has already been reported. 
2. **Provide detailed information**: When reporting a bug, include the following details:
   - Description of the bug
   - Steps to reproduce the issue
   - Expected behavior
   - Actual behavior
   - Screenshots/logs (if applicable)

## Feature Requests
1. **Check for existing features**: Before submitting a new feature request, ensure it is not already being considered.
2. **Submit a clear proposal**: Provide a clear description of the feature you would like to see, including:
   - Use case
   - Benefits of the feature
   - Any additional details that may be useful

## Pull Request Process
1. **Fork the repository**: Start by forking the repository to your account.
2. **Create a feature branch**: Create a new branch for your feature or bug fix. Example: `feature/my-feature` or `bugfix/my-bug`.
3. **Make your changes**: Implement your changes and ensure that they align with the project's code style.
4. **Write clear commit messages**: Write clear and concise commit messages to describe your changes.
5. **Open a pull request**: Submit a pull request to the **main** branch of the repository. Provide a clear description of the changes you made.
6. **Address any feedback**: Be open to feedback and make necessary changes to your pull request as requested by reviewers.

## Code Style Rules
- **Indentation**: Use 2 spaces for indentation.
- **Naming conventions**: Use camelCase for variable names and PascalCase for class names.
- **Commenting**: Write clear comments to explain complex logic or important processes.
- **Formatting**: Follow consistent code formatting practices. You can use linters or formatters as needed.

Thank you for your contributions! Let's build something great together!
